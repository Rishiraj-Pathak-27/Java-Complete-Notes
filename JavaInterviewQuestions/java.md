# Imperative & Functional Programming (Lambda Expression (->))

## Imperative Approach 

Doing anything itself
Eg. Searching for the table in a restaurent.

## Functional Approach

Calling someone to do the specific task
Eg. Hey waiter! Table for 5-6

## Syntax:

[   ( ) -> {  }   ]

where ( ) = Parameters
      -> = Lambda Operator
      { } = Lambda Body

When ever use use a single statement we use remove braces
